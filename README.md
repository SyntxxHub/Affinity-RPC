# Affinity RPC <img src="affinity_rpc.png" width="40" align="center">
Discord Rich Presence for Affinity Suite apps.

## What it does
Displays which Affinity app you're currently using as your Discord status.

Supports:
- Affinity Designer 2
- Affinity Photo 2
- Affinity Publisher 2

## Requirements
- Windows
- Discord (running)
- Any Affinity Suite app

## Usage
Simply run `AffinityRPC.exe` and it will appear in your system tray. Your Discord status will automatically update when you open any supported Affinity app, and clear when you close it.

## Notes
- Only shows which app is open, not the file you're working on
- Right-click the tray icon and click **Quit** to exit
